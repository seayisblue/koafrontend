# 轻度康复方案 (Level 4)

## 功能概述

实现了完整的轻度康复训练方案，包含两种训练方式：

### 1. 选择页面 (LightRehab)
- **路由**: `/light-rehab`
- **功能**: 展示两种康复方式供用户选择（配图展示，设计矢量图标）
  - A: 有氧运动 🏃
  - B: 神经肌肉训练 / 本体感觉训练 🧘

### 2. 有氧运动 (MildAerobic)
- **路由**: `/mild-aerobic`
- **训练说明**:
  - 低冲击性运动为主
  - 频率：每周 3-5 次
  - 强度：中等
  - 时长：每次 20～45 分钟
  - 小提示：达到最大心率的 60-80%，或主观感觉"有点吃力"，避免疼痛加重
- **功能**:
  - 展示训练说明和小提示
  - 点击【好的】进入打卡页
  - 提供三种运动选项打卡：
    - 快走 🚶
    - 室内健身自行车 🚲
    - 游泳 🏊
  - 点击后弹窗二次确认（参考留小白）
  - 完成后显示鼓励信息

### 3. 神经肌肉训练 / 本体感觉训练 (MildNeuro)
- **路由**: 
  - 介绍页：`/mild-neuro`
  - 方式选择页：`/mild-neuro/select`
  - 步骤要点页：`/mild-neuro/:exerciseId`
  - 视频播放页：`/mild-neuro/:exerciseId/video`
- **训练说明**:
  - 现在我们来做膝关节周围肌肉的肌力训练
  - 频率：每周 2-3 次
  - 强度：中等
  - 时长：8-12次/组，每个动作2-4组
  - 小提示：请避免出现呼吸困难、心悸等情况，避免出现明显疲劳或其他不适
- **功能流程**:
  1. 介绍页：展示训练说明，点击【好，开始吧】进入方式选择页
  2. 方式选择页：用户依据自身情况，多选以下运动训练
  3. 步骤要点页：展示具体训练的准备、步骤和小提示
  4. 视频播放页：播放训练视频，完成后打卡
- **8种训练方式**:
  1. **腘绳肌/股四头肌等长收缩**
     - 频率：10次/组，重复2-3组
     - 准备：坐在椅子上，弹力带分别固定于脚踝和椅子腿
     - 步骤：双腿弯曲，缓慢抬起一侧小腿，直到膝关节尽量伸直，在伸直的位置保持5秒，感受大腿前侧发力，有控制地慢慢放回原位
     - 视频：`knee_stretch_when_sitting`
  
  2. **上下台阶**
     - 上台阶步骤：先将患侧脚踩上第一级台阶，后脚缓慢且匀速跟上，控制患侧膝关节匀速从弯曲到伸直，双脚站在同一台阶后再将患侧踩上第二节台阶，按此节奏连续上3-5节台阶，动作连贯
     - 下台阶步骤：站最上面台阶，先抬健侧脚迈向下一级台阶，同时缓慢且匀速弯曲支撑侧的膝关节，待健侧脚完全放在下一级台阶时，另一只脚再放下至同一级台阶，双脚站在该级台阶后再次抬健侧脚迈向下一级台阶，按此节奏连续下完所有台阶，动作连贯
     - 小提示：始终以第一只脚主动发力上台阶，后脚辅助跟随；每步踩稳再动，步幅小、速度慢，膝盖不绷直；若膝盖疼或不稳，立即停下，减少台阶数量
     - 视频：`walk_up_and_down_the_stairs`
  
  3. **上下台阶训练（单个）**
     - 准备：选择高度适宜的台阶，穿着舒适防滑的运动鞋，身旁备好扶手辅助稳定
     - 上台阶步骤：先将患侧脚踩上台阶，后脚缓慢且匀速跟上，控制患侧膝关节匀速从弯曲到伸直，直至双脚站在同一台阶，过程中保持身体直立，避免前倾
     - 下台阶步骤：先抬健侧脚离开台阶，同时缓慢且匀速弯曲支撑侧的膝关节。待健侧脚完全放在下一级台阶时，另一只脚再跟随放下
     - 小提示：注意保护防止摔倒。每次仅上下1-2级台阶，动作缓慢匀速；避免膝关节发力蹬踏；若出现明显疼痛，立即停止
     - 视频：`walk_up_and_down_the_block`
  
  4. **前后滑动（平衡垫）**
     - 平衡垫购买建议：可以在京东/淘宝等正规在线商城，按情况搜索购买"瑜伽平衡垫"或"健身平衡垫"
     - 训练准备：取稳固座椅
     - 步骤：左脚站立于平衡垫，右脚自然着地，双手轻扶椅保持稳定。右脚缓慢向前滑动，同时左膝缓慢匀速弯曲膝关节至40°左右且稳定，然后右脚再缓慢向后滑动至起始位置，同时左膝由弯曲缓慢匀速至伸直为止。接着右脚缓慢向后滑动，同时弯曲膝关节至40°左右，然后右脚向起始位置返回，同时左膝由弯曲缓慢伸直
     - 小提示：注意保护患者，防止摔倒。动作匀速、小幅度。若关节刺痛、平衡失控，立即停止
     - 视频：`slide_forward_and_back_with_balancing_pad`
  
  5. **坐站转移训练（进阶版）**
     - 步骤：选择稳固的座椅（如带靠背的椅子，避免滑轮椅或高脚凳）。双脚可完全踩实地面，膝盖弯曲约90°。一只脚向后撤半步，另一只脚往前挪半步，双脚间距与肩同宽，身体微微前倾，重心缓慢移至后方的脚上，收紧大腿和臀部肌肉发力。双手向前抬起至与肩同高（约90°）保持平衡，缓慢站起，感受大腿发力，待身体稳定后，双手自然收回，收回前面的脚，慢慢坐下
     - 视频：`advanced_stand_to_sit`
  
  6. **单腿站立平衡（平衡垫）**
     - 平衡垫购买建议：可以在京东/淘宝等正规在线商城，按情况搜索购买"瑜伽平衡垫"或"健身平衡垫"
     - 步骤：取稳固座椅辅助，缓慢抬起健侧腿，单腿站立于平衡垫，保持身体平衡，维持10~20秒直至大腿感受到酸痛后缓慢放下，2-3组
     - 小提示：注意保护患者，防止摔倒
     - 视频：`stand_on_one_leg_with_balancing_pad`
  
  7. **单腿站立平衡（髋外展推墙）**
     - 步骤：侧身站立，健侧腿贴近墙面。双手背于身后，保持躯干稳定，患侧站立，抬起健侧后用靠近墙一侧腿向外侧轻推墙面，同时支撑腿微微下蹲（屈膝40°左右），维持姿势20秒，随后控制身体，缓慢伸直健侧腿起身，回到起始位
     - 频率：重复动作2-3组
     - 视频：`stand_on_one_leg_against_wall`
  
  8. **股四头肌牵伸**
     - 步骤：站立，单手扶墙，另一手抓住脚踝，将小腿向臀部方向拉，直到前侧大腿有牵拉感
     - 时长：保持15-30秒
     - 频率：重复2-3次
     - 视频：`quadriceps_stretching`

## 技术实现

### 页面结构
```
level4/
├── LightRehab/          # 选择页面（2种方式）
├── MildAerobic/         # 有氧运动
│   ├── MildAerobic.vue      # 介绍页
│   └── MildAerobicCheck.vue # 打卡页
├── MildNeuro/           # 神经肌肉训练
│   ├── MildNeuro.vue        # 介绍页
│   ├── MildNeuroSelect.vue  # 方式选择页（多选）
│   ├── MildNeuroDetail.vue  # 步骤要点页
│   └── MildNeuroVideo.vue   # 视频播放页
└── shared/              # 共享组件（预留）
```

### 核心功能
1. **视频播放**: 使用原生 `<video>` 标签，支持自动播放和控制
2. **路由导航**: 使用 `vue-router` 实现页面跳转
3. **状态管理**: 使用 `ref` 和 `watch` 管理视频播放状态和多选状态
4. **交互反馈**: 使用 `showActionSheet` 提供确认和鼓励信息（参考留小白）
5. **国际化**: 所有文本通过 `vue-i18n` 管理
6. **多选功能**: 神经肌肉训练支持用户多选训练方式

### 样式特点
- 响应式设计，适配移动端和桌面端
- 卡片式布局，清晰的视觉层次
- 使用 CSS 变量保持主题一致性
- 悬停和点击动画提升用户体验
- 矢量图标设计（选择页面）

1. `walk_up_and_down_the_stairs` - 上下台阶
2. `walk_up_and_down_the_block` - 上下台阶训练（单个）
3. `advanced_stand_to_sit` - 坐站转移训练（进阶版）
4. `stand_on_one_leg_with_balancing_pad` - 单腿站立平衡（平衡垫）
5. `stand_on_one_leg_against_wall` - 单腿站立平衡（髋外展推墙）
6. `quadriceps_stretching` - 股四头肌牵伸

已有视频可复用：

- `slide_forward_and_back_with_balancing_pad` - 前后滑动（平衡垫）

## 使用说明

### 有氧运动流程
1. 访问 `/light-rehab` 进入选择页面
2. 选择"有氧运动"进入介绍页
3. 阅读训练说明和小提示
4. 点击【好的】进入打卡页
5. 选择一种运动（快走/室内健身自行车/游泳）
6. 点击后弹窗二次确认
7. 完成后显示鼓励信息

### 神经肌肉训练流程
1. 访问 `/light-rehab` 进入选择页面
2. 选择"神经肌肉训练"进入介绍页
3. 阅读训练说明和小提示
4. 点击【好，开始吧】进入方式选择页
5. 多选训练方式（8种可选）
6. 点击具体训练方式进入步骤要点页
7. 阅读准备、步骤和小提示
8. 点击【好，开始吧】进入视频播放页
9. 观看视频并完成训练
10. 打卡确认

## 注意事项

- 视频文件需要上传到 OSS 并配置正确的 CDN 地址
- 视频文件命名需与 `config/files.ts` 中的配置一致
- 所有国际化文本需添加到 `zh-Hans.json`
- 删除了肌力训练和关节活动度训练相关代码和路由
- 神经肌肉训练从4个固定视频改为8种可多选的训练方式
